The Pond simulator is a tool for training and testing the Pond robots.

It is built with [rapier](https://rapier.rs/) and [Pond Bus](https://github.com/ecto/pond/blob/main/crates/bus/src/lib.rs).

## Why not Isaac Sim?

Isaac Sim is a great simulator, but it is not open source.

## Why not Gazebo?

Gazebo is a great simulator, but it is not open source.
