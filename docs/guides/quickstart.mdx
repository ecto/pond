<Steps>
  <Step title="Install prerequisites">
    <ul>
      <li><strong>OS</strong> ‚Äì Linux (Ubuntu 22.04) or macOS 14+. Windows works via WSL 2.</li>
      <li><strong>Rust toolchain</strong> ‚Äì install via <code>rustup</code>.</li>
      <li><strong>Nix</strong> (optional) ‚Äì guarantees matching compilers and Python versions.</li>
      <li><strong>Git &amp; GitHub</strong> ‚Äì to clone and contribute.</li>
      <li><strong>Basic shop tools</strong> ‚Äì drill press, deburrer, hex keys, soldering iron.</li>
    </ul>
  </Step>

  <Step title="Clone &amp; bootstrap">

```bash
# one-liner
git clone https://github.com/ecto/pond.git && cd pond
nix develop   # or install Rust + Python manually
```
  </Step>

  <Step title="Build the software">

```bash
cargo check
cargo run -p pad -- --profile deck-eco
```
  If a Bevy window pops up, you're set.
  </Step>

  <Step title="Generate CAD">

```bash
cargo run --bin generate-cad
```
  STEP &amp; STL files appear under <code>target/cad/</code>.
  </Step>

  <Step title="Order parts">
    Open the BOM in <code>notes/arm.md</code> (or <code>frog.md</code>) and load it into your favourite supplier. Prime shipping ‚âà robot next week.
  </Step>

  <Step title="Flash firmware">

```bash
cd crates/firmware
pio run -e pico
```
  Hold <code>BOOTSEL</code> then drag-and-drop the UF2 onto the RP2040.
  </Step>

  <Step title="First motion">
    1. Power on with E-Stop out.<br/>
    2. Launch <code>pad</code> and connect over USB/Wi-Fi.<br/>
    3. Release E-Stop and jog joints in <strong>Teleop</strong> view.
  </Step>
